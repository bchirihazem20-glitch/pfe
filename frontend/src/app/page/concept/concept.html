Concept de l’Académie Sportive

Ce site web a été conçu pour faciliter la gestion d’une académie sportive et améliorer la communication entre l’administration, les entraîneurs, les joueurs et les parents.

Grâce à cette plateforme, les utilisateurs peuvent s’inscrire en ligne, consulter leurs informations personnelles et suivre les activités sportives de l’académie.

Les entraîneurs peuvent gérer les joueurs, organiser les séances d’entraînement et suivre les performances des joueurs.

Le site est développé avec Angular pour l’interface utilisateur et Spring Boot pour le backend, avec un système d’authentification sécurisé basé sur JWT.
